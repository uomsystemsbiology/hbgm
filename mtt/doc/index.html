Model Transformation Tools

  • MTT comprises a set of tools for modelling dynamic physical systems using the bond-graph methodology and transforming these models into representations suitable for analysis, control and simulation. These tools use, and generate m files for, GNU Octave.
  • MTT is mainly built on the GNU set of tools.
  • MTT uses the algebraic package Reduce
  • A manual is available in html and pdf form.
  • MTT is based on the bond graph language; general bond graph information is available from the Bond Graph web site in Arizona
  • The background theory is given in a book and a tutorial paper.
  • Some examples generated by MTT are available.
  • Downloads, forums, bug reports etc are to be found here, courtesy of sourceforge.
